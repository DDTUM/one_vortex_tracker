# one_vortex_tracker
Алгоритм для отслеживания одного вихря в векторном поле скорости

**Задача: Отследить координаты вихря в векторном поле скорости.**
Основной код это `onevorttracker.m`

Также в есть пример полей скорости (_Frame_ex.mat_), которые служат для демонстрации работоспособности кода. Его можно получить по ссылке (https://drive.google.com/drive/folders/172GTVYK1z-JLSuFKj7Btnd1GX9RyNQWN?usp=sharing)
Кроме того, результат работы можно увидеть по ссылке: https://www.youtube.com/watch?v=Dfqi34WKLEI&feature=youtu.be



В коде присутствуют комментарии, тут тоже описана работа этого кода.

## Поиск центра вихря 
Первоначально необходимо задать первоначальные координаты вихря. Далее произойдет уточнение и поиск дальнейших центров перемещающегося вихря. За это ответственна функция 

`processing_find_center.m`

Алгоритм основан на поиске экстремума тангенциальной скорости.

```matlab
[xc,yc] =processing_find_center(xc, yc, num, frame, frameVel);

%% В коде встречается
[xc,yc] =processing_find_center(xc, yc, numbergrid, 1, temp, frameVel);

```
В функции _processing_find_center_ есть параметр `area=5;` в начале кода показывает, на какой площади будем искать центр вихря. Создается сетка поиска. Для каждой координаты этого пространства рассчитывается средняя тангенциальная скорость в радиусе 2,5 см при помощи алгоритма.
`f_Radial_profile_win2.m`

```matlab
[loss_mean] = f_Radial_profile_win2(frame, x0, y0,tim,rmax);
```

Далее ищется экстремум средней тангенциальной скорости. Экстремум, потому что знак зависит от направления. (А так желтые вихри -> -1 -> синие вихри -> вращение по часовой стрелке -> отрицательная тангенциальная скорость ->  поиск минимума тангенциальной скорости).

```matlab
% ind_ans = find(-fit == min(min(-fit))); %for @__> yellow
ind_ans = find(fit == min(min(fit)));  %for @__< blue
```

Таким образом и находится центра вихря с точностью до шага сетки PIV.
